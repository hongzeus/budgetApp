# budgetapp-cli

## 언제 사용하나
- CSV 기반 가계부 CLI 앱의 기능을 추가하거나 수정할 때 사용한다.
- 거래 조회, 합계, 필터, 정렬, CSV 입출력, 테스트 작성 작업에 적용한다.

## 핵심 원칙
- 먼저 테스트를 작성하고 구현한다.
- 모든 함수에는 타입 힌트를 넣는다.
- 함수는 50줄 이하로 유지한다.
- 순환복잡도는 10 이하로 유지한다.

## 작업 지침
- CSV 구조를 바꾸기 전에 기존 데이터 호환성을 확인한다.
- CLI 입력은 항상 검증하고, 실패 시 명확한 에러 메시지를 낸다.
- 변경 후에는 `pytest`와 `radon cc` 관점으로 다시 점검한다.

## 품질 검수
- 커밋 전에는 `qa_engineer` 서브에이전트 검수를 거친다.
